# Exposure Management Reference

This repository contains a reference of terms and concepts related to Exposure Management, organized to enable a fast, holistic, and practical understanding of the domain.

The content is also published as a static site at:  
➡️ https://exposure-notes.dev/index.html

## Purpose

The goal of this project is to serve as a living reference for concepts used in:

- Security solution evaluations  
- Daily operations (SecOps, posture management, exposure management)  
- Shared discussion and understanding of the domain  
- Conceptual and architectural analysis support  

## Content origin

The site and the content of this repository were built from:

- Terms gathered during solution evaluations, daily operational activities, and hands-on work within the security domain.  
- Organization and structuring with the help of Copilot, aiming to group concepts in a way that:
  - Enables a holistic understanding  
  - Facilitates quick reading  
  - Exposes natural relationships between concepts  

This process resulted in a nesting effect, where broader concepts contain and relate to more specific ones.

## Project evolution

I plan to:

- Add new terms  
- Expand descriptions  
- Incorporate additional context and references  
- Refine the structure as the domain evolves
