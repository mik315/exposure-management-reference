# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public issue.**

Instead, please email: **[tu-email-aquí]**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Response

I will acknowledge receipt as soon as possible and provide updates as the issue is investigated and resolved.

## Scope

Relevant vulnerabilities may include XSS via crafted URLs, dependency issues, or content injection.

## Thank You

I appreciate responsible disclosure and will credit reporters (with permission) in the project changelog.
